# 待办事项管理工具

一个功能完善的待办事项管理Web应用，支持多标签、前置条件、依赖关系等高级功能。

## 功能特点

- ✅ 多标签系统（复选框式选择）
- ✅ 前置条件管理
- ✅ 依赖关系可视化
- ✅ 状态循环切换
- ✅ 筛选功能（按责任人、DDL、标签、状态）
- ✅ 每日总结生成
- ✅ 过期事项提醒
- ✅ 响应式设计（支持手机访问）
